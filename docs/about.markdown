AstrophotoKit is a Swift package designed for astronomical image processing. It provides tools for working with FITS files, performing GPU-accelerated image processing using Metal shaders, and building pipelines for astronomical data analysis.

## Features

- **FITS File Support**: Fast FITS file reading/writing using CFITSIO
- **Metal Shaders**: GPU-accelerated image processing with custom Metal shaders
- **Swift API**: Clean, Swifty interface for astronomical data processing
- **Pipeline System**: Build complex image processing pipelines with reusable steps
